# <a name="securitynegotiationfailure"></a>SecurityNegotiationFailure
Id: 172  
  
 严重性：错误  
  
 类别：SecurityAudit  
  
## <a name="description"></a>描述  
 此事件指示安全协商失败。 事件将列出服务、操作、客户端标识、活动 ID 和协商。
