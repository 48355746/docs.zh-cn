# <a name="xml-comment-parameter-39ltparametergt39-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>XML 注释参数&#39;&lt;参数&gt;&#39;相对应的参数不匹配&lt;关键字&gt;语句
XML 注释参数\<参数 > 上相应的参数不匹配\<关键字 > 语句。 将忽略 XML 注释。  
  
 为方法中的某些参数（而不是全部）指定了 XML 注释标记。  
  
 **错误 ID：** BC42307  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为方法中的所有参数提供 XML 注释标记。
